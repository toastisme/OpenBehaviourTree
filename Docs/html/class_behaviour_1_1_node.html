◆ TaskName

string Behaviour.Node.TaskName
getset

If a decision node (SequenceSelector, ProbabilitySelector, PrioritySelector), TaskName is the node type.

If a decorator, TaskName is a boolean key in the BehaviourTreeBlackboard. If a ProbabilityWeight, TaskName is an int or float key in the BehaviourTreeBlackboard. If an ActionNode, TaskName is the name of a BehaviourTreeTask class.

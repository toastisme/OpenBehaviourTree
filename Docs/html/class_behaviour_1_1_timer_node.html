Detailed Description

Represents a timer in the BehaviourTree class (that things like timeouts and cooldowns can derive from) The timer runs for TimerValue +/- RandomDeviation.

If valueKey or RandomDeviationKey set, TimerValue and RandomDeviation are set using these keys whenever StartTimer() is called.
